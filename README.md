# KitUp legal site (GitHub Pages)

Static site for TikTok Developer Portal **Website URL**, Privacy Policy, Terms of Service, and OAuth callback.

**Live target:** https://bilalgurkansanli.github.io/kitup-legal/

## Deploy to `kitup-legal` repo

1. Clone or open your GitHub repo `bilalgurkansanli/kitup-legal`.
2. Copy **all files** from this folder into the repo root (`index.html`, `privacy.html`, `terms.html`, `callback.html`, `styles.css`).
3. Commit and push to `main` (or your Pages branch).
4. Wait 1–2 minutes, then check:
   - https://bilalgurkansanli.github.io/kitup-legal/ — must show KitUp home with Privacy/Terms in header
   - https://bilalgurkansanli.github.io/kitup-legal/privacy.html
   - https://bilalgurkansanli.github.io/kitup-legal/terms.html

## TikTok Developer Portal

| Field | URL |
|-------|-----|
| Website URL | `https://bilalgurkansanli.github.io/kitup-legal/` |
| Privacy Policy URL | `https://bilalgurkansanli.github.io/kitup-legal/privacy.html` |
| Terms of Service URL | `https://bilalgurkansanli.github.io/kitup-legal/terms.html` |
| Redirect URI | `https://bilalgurkansanli.github.io/kitup-legal/callback.html` |

Then use **Verify URL properties** for each URL (Production), and resubmit the app for review.

Demo video should open the **home page** (`/kitup-legal/`) so reviewers see Privacy/Terms links without opening a submenu.
